    1. Mortar Documentation

    Mortar Documentation

    Mortar is a Go framework/library for building gRPC (and REST) web services. It has out-of-the-box support for configuration, application metrics, logging, tracing, profiling, dependency injection and more.

    The key features are:

    • Dependency Injection using Uber-FX.
      • Everything is pluggable, however you can replace everything.
    • Middleware concept is almost everywhere and very customizable.
      • Build-in Server/Client Interceptors both for gRPC/HTTP, you can bring your own.
      • Telemetry - Metrics, Tracing and Logging all connected.
    • Increases Developer Velocity
    • DevOps friendly
      • Builder pattern
      • Profiling, Debug, Build Info, Configuration and more.

    The simplest way to start is to clone the Demo repository.
